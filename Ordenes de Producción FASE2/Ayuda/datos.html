Datos de Ordenes

No. Orden

Es un código asignado para la generación de nuevas ordenes, el cual no sera ignresado por el usuario si no que, se irá incrementado conforme a la generación de ordenes.


Código Producto

Es un código asignado para visualizar los diferentes productos existentes, el cual no sera ingresado por el usuario si no que, al darle clic al boton "?" que se encuentra junto al TextBox "Producto a fabricar", nos mostrara una lista de productos existentes y al seleccionar uno automaticamente se pondra el código de ese producto dentro del TextBox de Código Producto.


Producto a Fabricar

Nombre del producto que se desea fabricar, el cual no sera ingresdo por el usuario si no que, al darle clic al boton "?" que se encuentra junto al TextBox de Producto a Fabricar, nos mostrara una lista de productos existentes y al seleccionar uno automaticamente se pondra el nombre del producto que se fabricara.


Cantidad Producto

Cantidad a fabricar del producto deseado, esta cantidad si sera ingresada por el usuario ya que él es quien sabe cuanto producto se necesita o cuanto producto fue encargado.


Lote

Numero asignado a cada producto al entrar a los almacenes, el numero sera igual en dado caso sean productos iguales los que se fabricaran, el cual no sera ingresado por el usuario si no que, tendra un ComboBox que le ayudara a seleccionar el numero de Lote al que corresponde el producto.


Prioridad

Dependera de la urgencia que se posea el cliente y así saber con que Ordenes empezar, el cual no sera ingresado por el usuario si no que, tendra un ComboBox que le ayudara a seleccionar el nivel de Prioridad que desee.


Fecha de Inicio

Fecha en que sera iniciada la orden de producción que se realizo, esta fecha si sera ingresada por el usuario ya que él es quien conoce la prioridad de la Orden.


Visualización de Productos

Si la Orden de Producción lleva variedad de productos a fabricar, primeramente se estarán ingresando y visualizando en un DataGridView que se encuentra dentro de este Formulario, para luego ser visualizados en la Orden que se generara.
