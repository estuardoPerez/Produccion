Se le mostrará una interfaz donde usted deberá seguir el siguiente orden

  1. Primero se debe seleccionar el id del empleado, co lo que se obtendra el nombre y el apellido

  2. Luego en el estado del proceso se verificara si este esta disponible consultando el numero de operarios requeridos
  3. en el estado del empleado lo que observara es que si esta disponible, en proceso o está de vacaciones

  5. En el dataGrid se le mostrarán los datos insertados

La barra superior contiene los botones para cerrar y minimizar la ventana actual en la que se trabaja
